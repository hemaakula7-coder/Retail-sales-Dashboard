# Retail-sales-Dashboard
“This Retail Sales Analytics Dashboard is built in Power BI, analyzing sales, profit, discounts, and regional performance. The model includes sales transactions, product categories, customer segments, and regions. KPIs include Total Sales, Profit Margin, Year-over-Year Growth, and Discount Impact on Profit". 
“The dashboard shows a sales trend line, regional bar charts, category performance, and a discount vs. profit scatter plot".
